# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2023-07-15

### Added

- Rate limiting feature to prevent API abuse
- Caching system to improve performance and reduce API calls
- Webhook notifications for real-time updates on moderation results
- Usage analytics and reporting
- Enhanced license validation with real-time backend checks
- New configuration options for rate limiting, caching, and backend URL

### Changed

- Updated `ModerationConfig` interface to include new configuration options
- Modified `ContentModerationService` to incorporate new features
- Updated `LicenseService` to include real-time license verification
- Enhanced error handling to include rate limit errors

### Updated

- All documentation files (README.md, INSTRUCTIONS.md, API_DOCUMENTATION.md) to reflect new features and changes

## [1.0.0] - 2023-07-10

### Added

- Initial release of the AI Content Moderation System
- Content moderation for text, image, and video
- License key generation and verification
- Subscription management
- Support for different license types (Individual, Small Business, Enterprise, Pay-as-you-go)
- API documentation
- Usage instructions

### Changed

- Moved from a subscription-based system to a license key-based system

### Removed

- Direct subscription management from the package (now handled externally)
